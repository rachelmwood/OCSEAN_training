# OCSEAN_training
Training materials for Python training as part of the OCSEAN project conducted in December 2025.

Welcome to this step-by-step guide for learning Python! This course is designed for individuals with no prior programming experience, aiming to introduce you to the fundamentals of Python, one of the most popular and versatile programming languages today.

**Acknowledgements:** These materials were adapted from training courses developed by the [Jean Golding Institute, University of Bristol](https://www.bristol.ac.uk/golding/). The original resources were based on the materials written by [Christopher Woods](https://chryswoods.com/beginning_python/), and revised by [Matt Williams](https://milliams.com/courses/beginning_python/).

## Intended learning outcomes
By the end of this course, you will:
* Understand what a programming language is.
* Have the skills to create simple Python code.
* Know how to execute instructions, or commands, in Jupyter notebooks.
* Feel more confident to embark on your programming journey!

## What is Python and why is it useful?

Python is a programming language - a formal system of instructions used to create computer applications. We can use Python to as a calculator, to load and clean data, and run statistical analyses.



# Further Resources for Python
## Jean Golding Institute:
All of the material in the notebooks provided were based on courses published on the [Jean Golding Institutes course](https://bristol-training.github.io/courses.html) directory. The notebooks for Day 1 and 2 were based on the course [Introduction to Python (Part 1)](https://bristol-training.github.io/intro-python-1/) and the notebook for Day 3 was based on the course [Introduction to Data Analysis  in Python (Part 1)](https://bristol-training.github.io/data-analysis-python-1/).

The JGI has many many more courses if you want to expand your skills in python after this workshop, some good ones to start with are:

*  [**Introduction to Data Analysis in Python (Part 2)**](https://bristol-training.github.io/data-analysis-python-2/) - This is a more advanced course that follows on from Part 1 and teaches you how to fit models to your data using scikit-learn.

## w3 schools:
There are many online courses designed to help people learn Python from scratch, or learn how to use specific Python packages. [W3schools](https://www.w3schools.com/python/) has so many really helpful courses going over different aspects of Python, from the basics that we covered in this workshop, to using Python for machine learning. We really recommend looking at the following pages if you want to expand your Python skills further:
* [**Matplotlib Tutorial**](https://www.w3schools.com/python/matplotlib_intro.asp) - matplotlib is a Python module used for plotting and data visualisation. It is more versatile and lets you create more complex visualisations than seaborn and lets you take your plotting skills to the next level!
* [**Python Reference**](https://www.w3schools.com/python/python_reference.asp) - This is a great overview page for when you’ve forgotten something or need to refresh your memory of some more basic python functions and modules.

## Kaggle:
[Kaggle](https://www.kaggle.com/) is a resource full of datasets which are great for you to practice different data analysis tools on. They also often come with notebooks from other people so you can see what others have done with the data, and challenges to give you inspiration for tasks you might want to complete. Some examples of interesting datasets for you to play around with are:
* [**Spotify Global Music Dataset**](https://www.kaggle.com/datasets/wardabilal/spotify-global-music-dataset-20092025/code) - One of the most well-known music streaming services in the world, Spotify, offers comprehensive information about songs and artists. It offers insights on how music trends, artist popularity, and genres have changed throughout time and features both contemporary songs from 2025 and timeless hits from 2009 to 2023.
* [**World Top Restaurants Dataset**](https://www.kaggle.com/datasets/shaistashahid/world-top-resturants) - This meticulously curated dataset features 50 of the world's most acclaimed restaurants, spanning 19 countries across 5 continents. It provides comprehensive information about establishments that represent the pinnacle of culinary excellence, from Michelin-starred French institutions to innovative Nordic kitchens and contemporary Latin American cuisine.

## Online forums:
Sometimes you will get errors that you can’t find the root of and it can be tricky to solve. Luckily you probably won’t be the first! Forums can be a great way to gain wisdom from the wider coding community. You can search for solutions to your errors, or you can always post your own question! Some examples are:
* [**Stack Overflow**](https://stackoverflow.com/questions)
* [**The coding forums**](https://www.thecodingforums.com/forums/python.77/)

## AI Tools:
Some AI tools can be helpful for debugging or rewriting code. We really encourage you NOT to get it to write code by itself and always make sure you understand the output before using it. It can be most helpful when you get an error you don’t understand. When you have this issue you can always copy paste the exact error with the line of code that caused it.
